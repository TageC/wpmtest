# WPM Test - Typing Speed Test

A **minimalist, multi-language typing speed test** built with vanilla HTML, CSS, and JavaScript. No dependencies, no bloat—just pure typing practice.

## Features

✅ **Multi-Language Support**
- English (EN)
- Spanish (ES)
- Hindi (HI)
- French (FR)
- Mandarin Chinese (ZH)

✅ **Three Text Sources**
- Random Words (easiest)
- Literature (natural difficulty)
- Code Snippets (technical)

✅ **Customizable Duration**
- 15 seconds
- 30 seconds
- 60 seconds

✅ **Real-Time Statistics**
- WPM (Words Per Minute) - Net & Gross
- Accuracy Percentage
- Consistency Score
- Character Breakdown

✅ **Advanced Features**
- Horizontal scrolling text
- Blinking caret indicator
- Backspace across word boundaries
- Detailed results chart
- Keyboard shortcuts (Tab to restart)
- Mobile responsive
- Tab hint tooltip

✅ **Beautiful Design**
- Moss & Cream color palette
- Smooth animations
- Zero distractions
- Dark theme by default

## Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/wpmtest.git
cd wpmtest
```

### 2. Open in Browser
Simply open `index.html` in your browser:
```bash
open index.html
# or
start index.html
```

### 3. Start Typing!
- Click anywhere to focus
- Start typing to begin the test
- Press `Tab` to restart
- Results appear when time runs out
